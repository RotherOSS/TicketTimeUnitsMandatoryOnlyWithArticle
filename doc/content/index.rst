Sacrifice to Sphinx
===================

Description
===========
Inside the OTOBO standard system, the option Ticket::Frontend::NeedAccountedTime overrides the option Action::NoteMandatory. Therefore, it is not possible to configure dialogues without mandatory entry of articles and TimeUnits. This package changes the behaviour, TimeUnits are only mandatory if an article exists for it.
